### 📂 주요 폴더 기능

* **`blind_nav_system/` (내부 폴더)**: 로봇의 동작을 결정하는 **실제 파이썬 코드(.py)**들이 들어있는 가장 중요한 공간입니다.
* **`config/`**: 좌표나 설정값처럼 **자주 바뀌는 데이터**를 코드와 분리해서 보관하는 저장소입니다.
* **`launch/`**: 여러 개의 파이썬 파일(노드)을 일일이 켜지 않고, **한 번에 실행**시키기 위한 스크립트 보관함입니다.
* **`maps/`**: 로봇이 길을 찾을 때 참조하는 **지도 파일(.yaml, .pgm)**이 들어있는 곳입니다.

---

### 🐍 파이썬 코드 기능 (`blind_nav_system/` 내부)

| 파일명 | 주요 기능 (역할) |
| --- | --- |
| **`main_state_machine.py`** | **[전체 지휘]** 로봇의 '뇌'입니다. "인사하기 → 목적지 듣기 → 이동하기 → 도착 안내"로 이어지는 전체 순서를 관리합니다. |
| **`voice_interface.py`** | **[음성 소통]** 로봇의 '입과 귀'입니다. 사람의 말을 글자로 바꾸고(STT), 로봇이 할 말을 한국어 음성으로 출력(TTS)합니다. |
| **`navigation_client.py`** | **[주행 제어]** 로봇의 '다리'입니다. `main_state_machine`에서 받은 좌표로 로봇을 이동시키고, 도착했는지 실시간으로 확인합니다. |
| **`sensor_monitor.py`** | **[안전 감지]** 로봇의 '감각'입니다. 이동 중에 갑자기 나타난 장애물을 감지하거나, 사용자가 누르는 물리 버튼 신호를 체크합니다. |

---

### ⚙️ 설정 및 빌드 파일

* **`location.yaml`**: `A`, `B` 지점처럼 목적지의 이름과 실제 지도상의 좌표값을 매칭해둔 데이터 파일입니다.
* **`start_guide.launch.py`**: 터미널에서 이 파일 하나만 실행하면 위에서 말한 4개의 파이썬 노드가 동시에 작동합니다.
* **`setup.py`**: 작성한 파이썬 코드들을 ROS 2 시스템에 등록하여 `ros2 run` 명령어로 실행할 수 있게 해주는 연결 고리입니다.
